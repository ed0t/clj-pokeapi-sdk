by-id
(by-id id client)
Returns a stat based on id, nil if not found

by-name
(by-name name client)

from-pokemon
(from-pokemon pokemon client)
Resolve all the stats of a pokemon and returns the enriched collection.
If no stats are available an empty collection is returned
